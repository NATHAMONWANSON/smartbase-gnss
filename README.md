## Smart Base Station
### INTELLIGENT GNSS GROUND STATION FOR PRECISE POSITIONING AND IONOSPHERIC MONITORING

**Try It at :** https://smartbase_2025.jirapoom.work/d10/

**🚀 What We Do**

Our Smart GNSS Base Station is an advanced platform designed for high-precision positioning and environmental monitoring. Using Real-Time Kinematic (RTK) technology, we deliver centimeter-level correction data via the Internet (NTRIP) and radio (LoRa) for remote areas. Furthermore, we actively analyze ionospheric space weather parameters to visualize GNSS signal interference and manage user access all to ensure reliable, highly accurate satellite navigation under any conditions.

**✨ Key Features**

High-Precision RTK Positioning: Delivers centimeter-level accuracy by utilizing the Real-Time Kinematic (RTK) technique for GNSS signal correction.
Hybrid Data Transmission: Supports dual-mode communication to ensure reliability in all environments:
NTRIP Protocol: For real-time correction streaming via the Internet.
LoRa Radio: For high-range data transmission in remote or internet-inaccessible areas.
Advanced NTRIP Caster Management: Features a backend system for user authentication connection logging and historical auditing for service charge calculations.
Space Weather Web Dashboard: A comprehensive real-time platform that visualizes atmospheric parameters and trend graphs, allowing users to assess satellite navigation performance at a glance.

**🎯 Why It Matters**

Modern industries from smart agriculture to autonomous vehicles rely heavily on GNSS for precise positioning, but satellite accuracy is constantly threatened by the environment. Ionospheric disturbances can severely degrade signal precision, while remote locations often lack the internet connectivity required for standard error corrections. Our platform tackles these challenges head-on by delivering real-time, centimeter-level RTK corrections via both internet (NTRIP) and long-range radio (LoRa). By simultaneously monitoring space weather impacts, we empower users to maintain pinpoint accuracy and foresee signal disruptions under any condition.
